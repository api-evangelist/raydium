# Raydium (raydium)

Raydium is a leading Solana AMM and liquidity provider supporting standard AMM, concentrated liquidity (CLMM), and a launchpad. The Raydium API at `api-v3.raydium.io` exposes pool, farm, token, swap quote, and price endpoints.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/raydium/refs/heads/main/apis.yml)

## APIs
- **Raydium REST API v3** (`https://api-v3.raydium.io`) - public, free
- **Raydium Solana Programs** - on-chain AMM, CLMM, farm, launchpad

## Notes on OpenAPI
Raydium publishes Swagger UI at api-v3.raydium.io/docs/ but no canonical OpenAPI spec at a stable URL we could fetch.
